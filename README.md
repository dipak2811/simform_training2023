Steps to Run CRUD project:-

1) Need to clone git repository and checkout in Single-Page-Application-Using-Router branch

      git clone https://github.com/dipak2811/simform_training2023.git

      cd simform_training2023
      
      cd CRUD

      git checkout CRUD
